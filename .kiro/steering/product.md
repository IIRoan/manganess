# MangaNess Product Overview

MangaNess is a mobile manga reading application built with React Native and Expo. It provides a clean, ad-free reading experience for manga and manhwa with features including:

- Browse and search manga titles
- Read chapters with smooth vertical scrolling
- Track reading progress and bookmarks
- Dark mode support with customizable accent colors
- AniList integration for progress tracking
- Responsive design for various device sizes

The app focuses on user experience with performance monitoring, error boundaries, and accessibility features. It's designed as an educational project with emphasis on supporting original content creators.

## Key User Features

- Search and discovery of manga titles
- Chapter reading with WebView integration
- Reading progress tracking
- Bookmark management
- Theme customization (light/dark/system)
- Haptic feedback and smooth animations
